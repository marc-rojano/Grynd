
# Proyecto Grynd

Grynd es una aplicación web para gestionar entrenamientos, seguir el progreso y descubrir nuevas rutinas de gimnasio.

## Visión

La visión de Grynd es ser la plataforma de referencia para los entusiastas del fitness, proporcionando una experiencia de usuario intuitiva y potentes herramientas para ayudarles a alcanzar sus objetivos.

## Objetivos

- Crear una base de datos escalable para almacenar datos de usuarios, entrenamientos, ejercicios y progreso.
- Desarrollar un backend robusto con Node.js y Express.
- Construir un frontend responsive con React.
- Implementar la autenticación y autorización de usuarios.
- Permitir a los usuarios crear, ver, actualizar y eliminar sus entrenamientos y ejercicios.
- Proporcionar a los usuarios una forma de seguir su progreso a lo largo del tiempo.
- Ofrecer una biblioteca de rutinas de gimnasio predefinidas.
- Asegurarse de que el código esté bien documentado y sea fácil de mantener.

## Especificaciones Técnicas

- **Base de datos:** PostgreSQL
- **Backend:** Node.js, Express
- **Frontend:** React, TypeScript, Vite
- **Autenticación:** JWT (JSON Web Tokens)
- **Estilo:** CSS Modules, Emotion
